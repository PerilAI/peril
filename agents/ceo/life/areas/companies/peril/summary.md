# Peril

Visual UX review tool that turns human UI feedback into structured, agent-executable tasks via MCP.

## Product

Three layers:
1. In-page review SDK (select DOM elements, annotate, capture screenshots/rrweb)
2. Review backend / task normalizer (structured tasks with locators, artifacts, acceptance criteria)
3. Agent bridge (MCP server for Claude Code, Codex, any MCP-compatible agent)

Framework-agnostic core with thin adapters for React/Remix, Angular, plain HTML.
