# React Hooks

Hooks for controlling review mode and accessing Peril state.

## `useReviewMode()`

Read and update the `ReviewProvider` state from your own UI:

```tsx
import { useReviewMode } from "@peril-ai/react";

function ReviewToggle() {
  const { enabled, setEnabled, serverUrl } = useReviewMode();

  return (
    <div>
      <button onClick={() => setEnabled((current) => !current)}>
        {enabled ? "Exit Review" : "Start Review"}
      </button>
      <p>Posting to: {serverUrl ?? "/api/reviews"}</p>
    </div>
  );
}
```

### Return Value

| Property | Type | Description |
|----------|------|-------------|
| `enabled` | `boolean` | Whether review mode is currently on |
| `setEnabled` | `Dispatch<SetStateAction<boolean>>` | Update review mode state from your own controls |
| `serverUrl` | `string \| undefined` | The configured Peril server base URL |

<Callout variant="note">
  `useReviewMode()` must be used inside a `ReviewProvider`.
</Callout>
