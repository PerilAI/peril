# Screenshot Capture

`@peril-ai/core` exposes focused helpers for generating PNG screenshots of a
selected element or the current page.

## `captureElementScreenshot(element, options)`

Capture a single DOM element as a PNG data URL by default:

```typescript
import { captureElementScreenshot } from "@peril-ai/core";

const dataUrl = await captureElementScreenshot(buttonElement, {
  backgroundColor: null
});
```

### Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `backgroundColor` | `string \| null` | `null` | Override the screenshot background color |
| `format` | `"dataUrl" \| "blob"` | `"dataUrl"` | Return a base64 PNG data URL or a `Blob` |
| `html2canvasOptions` | `Partial<Html2CanvasOptions>` | `{}` | Forwarded to `html2canvas` |

## `capturePageScreenshot(options)`

Capture either the full document or the current viewport:

```typescript
import { capturePageScreenshot } from "@peril-ai/core";

const pageDataUrl = await capturePageScreenshot({
  scope: "document",
  format: "dataUrl"
});
```

`capturePageScreenshot()` also accepts `document`, `window`, `scope`, and the
same `backgroundColor`, `format`, and `html2canvasOptions` fields as
`captureElementScreenshot()`.
