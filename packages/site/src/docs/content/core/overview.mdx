# Core SDK Overview

`@peril-ai/core` is the framework-agnostic foundation of Peril. It handles element
selection, locator generation, screenshot capture, and review serialization.

## Key Concepts

### Locators

Peril generates a bundle of locators for each selected element, ordered by
resilience:

1. **testId** — `data-testid` attribute (most reliable)
2. **role** — ARIA role + accessible name
3. **css** — CSS selector
4. **xpath** — XPath expression
5. **text** — Text content match (least reliable)

### Reviews

A review captures everything an agent needs to understand and fix an issue:

- Element locators (multiple strategies)
- Screenshot of the selected element
- Full-page screenshot for context
- DOM snippet around the element
- Human comment describing the issue
- Page URL and viewport metadata

## Usage

```typescript
import {
  captureElementScreenshot,
  capturePageScreenshot,
  createReview,
  generateLocatorBundle,
  submitReview
} from "@peril-ai/core";

const locators = generateLocatorBundle(element);

const review = createReview({
  url: window.location.href,
  viewport: { width: window.innerWidth, height: window.innerHeight },
  selection: {
    boundingBox: { x: 24, y: 48, width: 160, height: 44 },
    locators,
    domSnippet: element.outerHTML
  },
  comment: {
    category: "polish",
    severity: "medium",
    text: "Button text is truncated on mobile",
    expected: "Keep the label on one line at tablet widths."
  },
  artifacts: {
    elementScreenshot: await captureElementScreenshot(element),
    pageScreenshot: await capturePageScreenshot()
  },
  metadata: {
    userAgent: navigator.userAgent,
    scrollPosition: { x: window.scrollX, y: window.scrollY },
    devicePixelRatio: window.devicePixelRatio
  }
});

await submitReview(review, {
  endpoint: "http://127.0.0.1:4173/api/reviews"
});
```

The core package is intentionally low-level: it focuses on locator generation,
review serialization, screenshot capture, and transport primitives. The React
adapter builds on top of these functions.

<Callout variant="tip">
  The core package has zero framework dependencies. Use it with React, Vue,
  Svelte, or vanilla JavaScript.
</Callout>
